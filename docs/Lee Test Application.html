Configuration file: configuration.eterm

Configuration file is an Erlang term that can be in either proplist:

{key1, Val1}.
{key2, Val2}.
...

or map form:

#{ key1 => Val1
 , key2 => Val2
 }.

Valid keys:

quux

This value controls quuxing, see: [quux]

xizzy

This value controls xizzying, see: [xizzy]
